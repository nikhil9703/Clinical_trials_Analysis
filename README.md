**Clinical_trials_Analysis**
Working on clinical trials data regarding the data cleaning ,data transformation and data visualization.

I have used the Clinical trails dataset from kaggle that has been mentioned in resources.

**Objective**
1. Understanding the Clinical trials Dataset.
2. Cleaning and preparing raw clinial trials dataset.
3. Identifying the pattern in the Dataset.
4. Creating Analysis based on it.


**Columns used Clinical Trials**

clin_trials <-- Dataset

clinical_trails_understanding_and_data_cleaning <-- ipynb file containing the data cleaning and transformation process 

clinical_trails_Analysis_and_chart <-- ipynb file containing the Analysis regarding the dataset
